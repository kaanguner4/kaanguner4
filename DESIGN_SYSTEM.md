# Profile Design System

## Palette

- Paper: `#F7F5F1`
- Light paper: `#FBFAF7`
- Ink: `#111111`
- Secondary ink: `#242424`
- Muted: `#7A746C`
- Divider: `#D8D2CA`
- Accent: `#9A8676`

## Principles

1. Monochrome first.
2. Use whitespace instead of decoration.
3. Origami objects should remain geometric and restrained.
4. Avoid external statistic-card services.
5. Use local SVG assets so the profile stays stable.
6. Keep claims factual; do not display invented GitHub statistics.
